# Pmod_ADC
8チャネルのADコンバータ・チップ MCP3008/3208をPmodで使うボード．  
8チャネルのアナログ信号を入力できます．  
チャネル1（ch1）については，テストピンもしくはPhone Jackからも入力できます．
